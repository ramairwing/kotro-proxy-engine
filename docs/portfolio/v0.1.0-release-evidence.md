## Executive summary

Korto Proxy Engine is an open-source, single-binary local AI reverse proxy that intercepts OpenAI and Anthropic streaming traffic. It provides semantic SSE caching, PII redaction, and context compression without a hosted SaaS gateway.

v0.1.0 ships a production-grade **four-channel distribution matrix** (GitHub Releases, npm, VS Code Marketplace, Homebrew) driven by an automated tag-triggered CI pipeline.

## Release pipeline architecture

```
git tag v*  →  GitHub Actions
                 ├─ matrix build (4 platforms)
                 ├─ consolidate → release-assets/*.tar.gz + .vsix
                 ├─ npm publish (@kortosystems/proxy-engine)
                 ├─ vsce publish (kortosystems.kortolabs-proxy-engine)
                 └─ softprops/action-gh-release
                        ↓
              scripts/update-homebrew-shas.sh v*
                        ↓
              ramairwing/homebrew-tap (formula + SHA-256)
```

Post-release automation: `make post-release-homebrew VERSION=v0.1.0`
